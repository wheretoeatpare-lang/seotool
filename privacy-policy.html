📋 Overview

🔒
Bottom line: we collect absolutely nothing. The RankSorcery SEO Analyzer extension performs all analysis locally inside your browser. No page content, URLs, analysis results, or personal information is ever sent to our servers or any third party.

This Privacy Policy describes how RankSorcery ("we", "us", or "our") handles information in connection with the RankSorcery SEO Analyzer browser extension, available for Chrome, Firefox, Edge, Opera, and Safari.

This policy applies to the browser extension only. For the privacy policy of the RankSorcery website (ranksorcery.com), please refer to the main site policy.

🔑 Permissions Explained

The extension requests the following browser permissions. Each is explained in plain language:

Permission Why It's Needed What We Do NOT Use It For
activeTab Read the HTML of the tab you are currently viewing when you click "Analyze Page" We do not read tabs you haven't clicked Analyze on, and never read tabs in the background
scripting Inject the SEO analysis function into the active page to extract meta tags, headings, schema, and performance data We do not inject tracking scripts, ads, or any persistent code into pages
storage Save your UI preferences locally (e.g., which sections were open) so the extension remembers your layout We do not store URLs, analysis results, or personal data in storage
<all_urls> Allow the extension to run on any website so you can analyze any page you visit We do not monitor your browsing or run any code on pages you haven't actively chosen to analyze
Note for Chrome Web Store reviewers: The <all_urls> host permission is required so users can analyze any website of their choosing. The extension only activates when the user explicitly clicks "Analyze Page" — it does not run automatically on page load or in the background.
04

⚙️ How the Extension Works

When you click "Analyze Page", the following happens entirely within your browser:

  • A JavaScript function is injected into the current active tab
  • The function reads HTML meta tags, heading structure, image attributes, link counts, schema markup scripts, and performance timing data from the page's DOM
  • This data is passed back to the extension popup and displayed to you
  • The data is held in memory only for the duration of the popup being open
  • When you close the popup, all page data is discarded from memory
  • Nothing is sent to any server — no network requests are made containing page data

The only outbound network requests the extension makes are:

  • Loading the Google Fonts stylesheet for the popup UI typography (fonts.googleapis.com) — this is a standard CDN request and contains no user or page data
  • Opening ranksorcery.com in a new tab when you click "Full Audit" or the footer link — this is a normal browser navigation, not a data transmission

06

💾 Data Storage

The extension uses the browser's local storage API (chrome.storage.local) solely to save UI preferences — specifically which analysis sections the user has expanded or collapsed. This data:

  • Never leaves your device
  • Contains no personal information, URLs, or analysis results
  • Can be cleared at any time by removing the extension
  • Is not synced to any cloud or external server

No cookies are set by the extension. No IndexedDB, sessionStorage, or other persistent storage mechanisms are used.
